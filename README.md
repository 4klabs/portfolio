# Portfolio project

## Usage

You need to use the `make` command and have `docker` and `docker-compose` installed first.

### Create environment variables

```sh
make setup
```

### Start app

```sh
make start
```

### View logs

```sh
make logs
```

### Use terminal

```sh
make terminal # or just "make"
```

### Stop app

```sh
make stop
```
